결제카드 유의사항

  • 결제카드는 최대 5개까지 등록 가능합니다.(네이버페이 포함)
  • 지역화폐 카드, 충전형 카드 등록이 불가합니다.
  • 등록된 카드로 결제 시 리터당 2P 적립됩니다.
  • 화물복지 카드 이용 시 바로할인 불가 합니다.
  • 카드 등록 시 100원 결제 후 자동 취소 됩니다.
  • 카드 청구 할인은 카드별로 다를 수 있으니 카드사에 문의하시기 바랍니다.
  • 신용카드 할부개월은
    금액설정 시 60,000원 이상
    리터설정 시
    • 휘발유 : 50리터 이상
    • 경유 : 50리터 이상
    • 고급휘발유 : 50리터 이상
    • LPG : 70리터 이상
    이상 시에만 할부선택이 가능하며 신용카드 승인 시 60,000원 미만의 경우 할부개월수 적용이 제외됩니다.
